Using python to create digital object manifest
•	Install python from software center
•	Download directories to local computer
•	Download create_csv_with_title.py
•	Place create_csv_with_titles.py alongside the directories you want to create manifest for
•	Create csv file with title and and source_metadata_identifier from file given by DCS
•	Open PowerShell
•	Navigate to directory with create_csv_with_titles.py
•	Run ‘py create_csv_with_titles.py’
•	Supply requested ‘parents’, ‘related url’, and ‘purl prefix’
